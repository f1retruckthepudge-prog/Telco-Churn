📊 Прогнозирование оттока клиентов телеком-компании
Telco Customer Churn Analysis
Цель: Выявить ключевые факторы оттока и предложить меры по удержанию клиентов на основе данных.

Инструменты: SQL (PostgreSQL), pgAdmin, Python (для модели, опционально).

1. Данные
Источник: IBM Telco Customer Churn

Строк: 7 043 клиента

Целевая переменная: Churn (Yes/No)

2. Общая картина
Показатель	Значение
Всего клиентов	7 043
Ушли (Churn = Yes)	1 869 (26.5%)
Средний срок жизни	32.4 месяца
Средний ежемесячный платёж	$64.8
Каждый четвёртый клиент покидает компанию. Необходимо найти управляемые рычаги для снижения оттока.

3. Ключевые инсайты
🔴 Сегмент с максимальным оттоком
Помесячный контракт + Оптоволокно (Fiber optic) — здесь отток достигает 54.6%.

Контракт	Интернет	Остались	Ушли	% оттока
Month-to-month	Fiber optic	966	1 162	54.6%
Month-to-month	DSL	829	394	32.2%
One year	Fiber optic	435	104	19.3%
Two year	Fiber optic	398	31	7.2%
Вывод: Долгосрочный контракт снижает отток в 7 раз. Помесячные клиенты на оптоволокне — самая горячая точка.

🛡 Безопасность и поддержка как фактор удержания
Рассмотрим только пользователей Fiber optic:

Сервис	Есть/Нет	Ушли	% оттока
OnlineSecurity	Нет	1 114 из 2 257	49.4%
OnlineSecurity	Да	183 из 839	21.8%
TechSupport	Нет	1 101 из 2 230	49.4%
TechSupport	Да	196 из 866	22.6%
Вывод: Отсутствие базовой кибербезопасности и техподдержки удваивает вероятность ухода.
Клиенты без этих сервисов не чувствуют связи с провайдером.

💸 «Дорогие» клиенты уходят быстрее
Среди помесячных контрактов:

Статус	Средний срок (мес.)	Средний чек
Остались	17.6	$61.3
Ушли	8.5	$72.9
Вывод: Ушедшие клиенты платили на $11.6/мес. больше, но прожили в 2 раза меньше. Потеря высокодоходных клиентов на раннем этапе — двойной удар по LTV.

🧾 Способ оплаты — маркер лояльности
Способ оплаты	Остались	Ушли	% оттока
Bank transfer (automatic)	1 286	258	16.7%
Credit card (automatic)	1 290	232	15.2%
Electronic check	1 294	1 071	45.3%
Mailed check	1 304	308	19.1%
Вывод: Ручная оплата Electronic check — сильнейший предиктор оттока. Автоматические платежи = лояльность.

4. Итоговый портрет «уходящего клиента»
Контракт: Month-to-month

Интернет: Fiber optic

Без OnlineSecurity и/или TechSupport

Платит через Electronic check

Высокий чек (~$73), но малый срок жизни (8–9 мес.)

Мера	Цель	Ожидаемое снижение оттока в сегменте	Доп. сохранённые клиенты / год
